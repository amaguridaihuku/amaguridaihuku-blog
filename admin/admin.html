アナリティクス（GoatCounter）

無料のGoatCounterを使うと訪問者数・人気ページが「📊 アナリティクス」タブで確認できます。

1
アカウント登録（無料）

goatcounter.com で「Sign up」→ サイトコードを決める（例: amaguridaihuku）

2
APIトークンを取得

ログイン後 → Settings（右上アイコン）→ API → 「Create token」

3
下の欄に入力して保存

保存するとブログにも計測スクリプトが自動で追加されます

サイドバーに表示される即売会・イベント情報を管理します。
